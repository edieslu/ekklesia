Se denomina accesskey a aquellas combinaciones de teclas que permiten acceder a diferentes lugares del sitio Web utilizando únicamente el teclado. El accesskey se compone de una tecla específica, dependiente del navegador, y de una tecla de función, número o letra.

Para hacer uso de los accesskey mediante el navegador deberá:

  • Pulsar sin soltar la tecla Ctrl
  • Pulsar la tecla de función, letra o número adecuado para la opción seleccionada
  • Soltar la tecla Ctrl y
Crtl + F1 = Inicio F6 y F12 no funcionan en firefox F7 y f12 no funciona en opera F12 no funciona en chrome Ctrl + i no funciona en firefox ctrl + b no funciona en opera ctrl + m no funciona en opera ctrl + l no funciona en opera
